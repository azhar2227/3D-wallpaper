# 🛒 3D wallpaper - ऑनलाइन शॉपिंग वेबसाइट

**3D wallpaper ** एक पूरी तरह से काम करने वाली ऑनलाइन शॉपिंग वेबसाइट है, जिसे HTML, CSS और JavaScript का उपयोग करके बनाया गया है। यह प्रोजेक्ट विशेष रूप से सीखने और छोटे व्यवसायों के लिए डिज़ाइन किया गया है।

## ✨ विशेषताएँ (Features)

- 🛍️ **प्रोडक्ट लिस्टिंग** — सुंदर ग्रिड लेआउट में प्रोडक्ट दिखाता है।
- 🛒 **कार्ट सिस्टम** — प्रोडक्ट्स को कार्ट में जोड़ें, मात्रा देखें, और हटाएँ।
- 💾 **लोकल स्टोरेज** — ब्राउज़र बंद करने पर भी कार्ट डेटा सेव रहता है।
- 💳 **चेकआउट फॉर्म** — ग्राहक से डिलीवरी जानकारी लेता है और ऑर्डर कन्फर्म करता है।
- 📱 **रिस्पॉन्सिव डिज़ाइन** — मोबाइल, टैबलेट और डेस्कटॉप पर सुंदर दिखता है।
- ⚡ **तेज़ और हल्का** — बिना किसी भारी फ्रेमवर्क के, सिर्फ शुद्ध वेब तकनीक।
- 🆓 **100% मुफ़्त और ओपन सोर्स** — कोई लाइसेंस शुल्क नहीं, MIT लाइसेंस के तहत।
